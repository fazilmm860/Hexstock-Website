1. Introduction <a name="introduction"></a>
The Hexstock website is a responsive e-commerce platform designed to showcase and sell a variety of products. 
It provides an intuitive user interface,displaying products with detailed descriptions and prices.
The website also features a blog section and a newsletter subscription option.

2. Project Structure <a name="project-structure"></a>
The project follows a standard HTML structure and uses Bootstrap for styling. 
It includes the following major sections:

Navigation: A responsive navbar with links to Home, Shop, Blog, Page, and Contact pages. It also includes search and cart icons.

Main Section:

Hero Container: Displays a prominent image with a tagline and a "Discover Me" button.

Responsive Div: Shows a selection of products in a responsive grid layout.

Sections:

Trending Products: Displays a selection of trending products with images, titles, and prices.

Product Essentials: Highlights essential products with images, titles, descriptions, and prices.

Sale Off: Features discounted products with images and prices.

Image Carousel: Displays a carousel of images.

Blog Posts: Shows blog posts with images, titles, authors, publication dates, and excerpts.

Subscribe Section: Offers a subscription form for the newsletter.

Footer: Contains information about the company, navigation links, and social media icons.
